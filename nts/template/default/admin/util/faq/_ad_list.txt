広告管理全体の仕組みについては『【ナパタウン】アフィリエイト連携資料』を一読ください。

広告管理
+ 広告管理について
広告をユーザ画面に表示してアフィリエイトを提供するために、各種パラメータを設定できます。広告のTOPページ（広告ポータル）の編集もできます。

+ 広告はユーザ画面のどこに反映されますか？
テンプレートファイルまたは静的HTMLファイルに記載された箇所全てに反映されます。管理画面で設定した形式のURLとパラメータをHTML形式で記載する必要があります。

+ 広告ASPとは何ですか？
アフィリエイトを提供しているASPまたは外部メディアのことです。広告ASPとしてASPコード名、ASPコードパラメタ、各種のパラメタを指定することでアフィリエイトが可能となります。

+ ASPコード名、ASPコードパラメタとは何ですか？
広告ASPを示す名称とコードのことです。ASPコード名とASPコードパラメタとで一対です。ASPコード名は任意の名称を、ASPコードパラメータはユニークな識別子として半角英数字で指定します。

+ 広告ポータルCMSとは何ですか？
広告のTOPページ全体を編集する機能のことです。広告ポータルの上部HTML・中部HTML・下部HTMLそれぞれに任意のデザインをHTML形式で指定します。

+ 広告ポータルCMSの端末振り分けとは何ですか？
ナパタウンでは機能しません。指定しても何も作用しません。

+ 広告カテゴリとは何ですか？
広告を構造種類別に分割するための識別名のことです。広告カテゴリ優先度として、ユーザ画面に表示される順番を「0」以上の整数で指定します。数字が小さい順に画面の上から表示されます。

+ IDとは何ですか？
広告毎に指定されているIDのことです。広告登録時にシステム側でユニークな整数が付与されます。管理画面からの変更はできません。

+ ステータスとは何ですか？
広告の状態のことです。広告が管理画面から登録されると有効に、削除されると無効になります。
削除した広告を再度有効にすることはできませんので、再度有効にしたい場合は同じ内容を新たに登録する必要があります。

+ バナータグとは何ですか？
広告自体を示すタグのことです。広告登録時にシステム側で付与されます。管理画面からの変更はできません。

+ リンク先URLとは何ですか？
広告を実際に記載する際のコードのことです。「ad.php?ad_id=〜」の内容をHTML形式で記載することで広告のリンク先の指定ができます。

+ 広告登録・広告編集におけるDOCOMO広告URL、au広告URL、SoftBank広告URLとは何ですか？
広告を実際に記載する際のコードのことです。キャリア（DOCOMO、au、SoftBank）毎に「http://〜??user=%%uaid%%〜」の内容をHTML形式で記載することで広告の指定ができます。

+ 広告登録・広告編集における広告取得ポイントとは何ですか？
広告で取得するポイント数（ナパポ）または料率広告タイプがワンクリックまたは登録の場合は固定値として「ナパポ」を、購買の場合は「%」を指定します。
備考はメモと同じで管理画面上のみに反映されます。

+ 広告登録・広告編集における広告タイプとは何ですか？
アフィリエイト広告の種別のことです。ワンクリック、登録、購買のいずれかを指定します。

+ 広告登録・広告編集における広告表示タイプとは何ですか？
広告を表示する種類のことです。サイト内表示（WEB）またはメール内表示（MAIL）のいずれかを指定します。

+ 広告登録・広告編集における広告配信終了数とは何ですか？
広告配信を終了する最大数のことです。指定された数まで配信を行い、この数を過ぎるとユーザ画面には表示されなくなります。

+ 広告登録・広告編集における広告配信開始日時（または広告配信終了日時）とは何ですか？
広告配信を開始（または終了）する日時のことです。指定された開始日時から終了日時まで配信を行い、終了日時を過ぎるとユーザ画面には表示されなくなります。

+ 広告登録・広告編集における広告配信開始日時設定（または広告配信終了日時設定）とは何ですか？
チェックボックスを有効にして、年月日時分秒を「0000-00-00 00:00:00」の形式で指定することにより、広告配信開始（または終了）日時を設定します。
