+ SNS基本情報管理について
ナパタウン全体の基本的な情報として、共通の名称、お知らせ、運用、色等の各種設定を行うことができます。

+ お知らせはどこに反映されますか？
ユーザ画面のマイページの最上位に表示されます。絵文字も利用できます。

+ 公開設定とは何ですか？
ナパタウン全体の公開・非公開の設定のことです。通常は公開とします。非公開とするとサイトの表示が行われません。

+ META Descriptionとは何ですか？
HTMLの<meta>タグのdescriptionの設定のことです。サイトにどんな内容が書かれているかを設定します。

+ META Keywordsとは何ですか？
HTMLの<meta>タグのkeywordsの設定のことです。サイトの内容を示すキーワードを複数設定します。

+ META robotsとは何ですか？
HTMLの<meta>タグのrobotsの設定のことです。検索エンジンや他の検索ロボットが検索する際の振舞いを設定します。
「noindex,nofollow」（推奨値）の設定の意味は、検索エンジンや他の検索ロボットがページのインデックス化をせず、リンクも辿らない、の意味となります。

+ META authorとは何ですか？
HTMLの<meta>タグのauthorの設定のことです。文書の制作者を設定します。

+ ナビゲーションテンプレートとは何ですか？
ユーザ画面各ページの下部に表示されるナビゲーションを促すページの設定のことです。HTML形式で設定します。

+ アクティブリンク色とは何ですか？
リンクを設定したテキストをクリックした時の色の設定のことです。

+ ページャ文字色とは何ですか？
一覧表示等の画面で表示されるページ制御「≪ Previous 1 2 3 .. Next ≫」で表示される箇所の色の設定のことです。

+ 端末振り分けとは何ですか？
端末機種が乏しい下位端末（例えばDOCOMOでデコメール機能が使えない端末）に対して提供するサービスを切り分ける設定のことです。キャリア（DOCOMO、au、SoftBank）毎に機種名を改行区切り（一行に一機種名）で設定します。
