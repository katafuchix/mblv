ゲーム管理
+ ゲーム管理について
ゲームに係る各種パラメータを設定を行うことができます。

+ ゲームカテゴリとは何ですか？
管理者が任意に設定できるゲームの種類のことです。ゲームを種類別に区分け・分類するために指定します。

+ ゲームカテゴリ優先度とは何ですか？
ゲームカテゴリのユーザ画面に表示される順番のことです。「0」以上の整数で指定します。数字が小さい順に画面の上から表示されます。「0」の場合は表示されません。

+ 一覧表示用ファイル、詳細表示用ファイルとは何ですか？
ゲームの表示に使用するファイルのことです。一覧表示用ファイルにサムネイル用を、詳細表示用ファイルに拡大用をそれぞれ指定します。

+ ゲーム消費ポイントとは何ですか？
ユーザがゲームを購入するときに必要となるポイント数のことです。整数で指定します。ユーザはこのポイント数以上を保持していなければ購入することができません。

+ ゲームカテゴリIDとは何ですか？
ゲームカテゴリ毎に指定されているIDのことです。ゲームカテゴリ登録時にシステム側でユニークな整数が付与されます。管理画面からの変更はできません。

+ ゲーム配信終了数とは何ですか？
ゲーム配信を終了する最大数のことです。指定された数まで配信を行い、この数を過ぎるとユーザは購入できなくなります。

+ ゲーム配信終了数設定とは何ですか？
チェックボックスを有効にして、整数で指定することにより、ゲーム配信終了数を設定します。

+ ゲーム配信終了日時とは何ですか？
ゲーム配信を終了する日時のことです。指定された日時まで配信を行い、この日時を過ぎるとユーザは購入できなくなります。

+ ゲーム配信終了日時設定とは何ですか？
チェックボックスを有効にして、年月日時分秒を「0000-00-00 00:00:00」の形式で指定することにより、アバター配信終了日時を設定します。

+ ゲームポータルCMSとは何ですか？
ゲームのTOPページ（ゲームポータル）全体を編集する機能のことです。ゲームポータルの上部と下部それぞれに任意のデザインをHTML形式で指定します。

+ ゲームポータルCMSの端末振り分けとは何ですか？
端末機種が乏しい下位端末（例えばDOCOMOでゲーム機能が使えない端末）に対して提供するサービスを切り分ける設定のことです。キャリア（DOCOMO、au、SoftBank）毎に機種名を改行区切り（一行に一機種名）で設定します。

+ ファイル管理とは何ですか？
「ナパタウン ファイルアップロード管理画面」が別ウィンドウで起動されます。フリーページに記載したい画像ファイルのアップロードやタグの表示ができます。
