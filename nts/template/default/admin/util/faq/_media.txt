+ 入会経路管理全体の仕組みについては『【ナパタウン】アフィリエイト連携資料』を一読ください。

+ 入会経路管理について
外部サイト（入会元メディア）からナパタウンへ入会（アフィリエイト入会）された時に、どのサイトからの入会なのかを識別するために、各種パラメータを設定等の操作を行うことができます。

+ 識別コードとは何ですか？
入会経路として入会元メディアに伝えるURLに設定されるパラメータのことです。「http://napatown.jp/index.php?code=〜」の内容をユニークな識別子として半角英数字で指定します。

+ 入会経路名とは何ですか？
入会経路を示す名称のことです。識別コードと入会経路名とで一対です。入会経路名は任意の名称を設定します。

+ 入会通知用メールアドレスとは何ですか？
アフィリエイト入会された場合にユーザへ送信するメールの送信元メールアドレスのことです。「nts-mn_〜@napatown.jp」の形式で識別コードに従ったメールアドレスが自動設定されます。

+ パラメータ名とは何ですか？
入会元メディアに入会成果を送信する際に必要となるパラメタ名のことです。パラメータ1、パラメータ2、パラメータ3の3個まで、それぞれ半角英数字で指定することができます。

+ 入会時成果返却先URLとは何ですか？
入会元メディアへ入会成果を送信する先のURLのことです。入会元メディア経由で受け取ったパラメータを入会時に指定URLに返却することができます。

+ 入会時参加コミュニティIDとは何ですか？
入会時に自動的に参加されるコミュニティIDのことです。入会元メディアと関連したコミュニティを登録することにより、コミュニティの活性化を図ることができます。半角数字で指定します。

+ 入会時付与ポイントとは何ですか？
入会時に自動的に付与されるポイント（ナパポ）のことです。「0」以上の整数で指定します。

+ パラメータ1、パラメータ2、パラメータ3とは何ですか？
入会元メディアからの入会時にシステム内で引き継ぐ情報のことです。入会成果を送信する際に必要となるパラメタ名を指定して下さい。

+ 入会経路通知メールタイトルとは何ですか？
アフィリエイト入会された場合にユーザへ送信するメールのタイトルのことです。

+ 入会経路通知メール本文とは何ですか？
アフィリエイト入会された場合にユーザへ送信するメールの本文のことです。メールがテキスト形式のため、HTML形式ではなく、テキスト形式で指定します。
