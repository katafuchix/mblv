ナパポ管理
+ ナパポ管理について
ユーザが取得または消費したナパポの履歴の確認を行うことができます。

+ ポイントIDとは何ですか？
ナパポ取得・消費毎に指定されているIDのことです。フリーページ登録時にシステム側でユニークな整数が付与されます。管理画面からの変更はできません。

+ ポイントメモとは何ですか？
該当ナパポに係る情報（アバター名やゲーム名等）のことです。

+ ナパポタイプとは何ですか？
ナパポの種類のことです。
SNS管理調整、クリック広告、登録広告、料率広告、アバター、デコメール、ゲーム、サウンド、入会経路、SNS会員登録、SNS友達招待(招待)、SNS友達招待(被招待)、EC管理調整、EC会員登録、EC購買があります。

+ ユーザサブIDとは何ですか？
広告のユーザ識別番号のようなユーザIDに係る補助的なIDのことです。

+ ポイントサブIDとは何ですか？
広告の広告ID等のようなナパポIDに係る補助的なIDのことです。

+ ポイントステータスとは何ですか？
該当ナパポの状態のことです。

+ ポイントステータスの却下、未承認、承認済みとはそれぞれどのような状態ですか？
以下の通りです。
却下　　　…　システムが却下した状態／ナパポに加算されません
未承認　　…　システムからの承認待ちの状態／まだナパポに加算されません
承認済み　…　システムが自動的に承認した状態／ナパポに加算されています

+ ポイントステータスはどのタイミングで決まるのですか？
入力経路や広告の設定内容により決まります。自動承認されて即時承認される場合や運用者からの承認待ちの状態となります。

+ 管理者がユーザの保持しているナパポを変更することはできますか？
ユーザ情報編集でナパポの値を変更することによってSNS管理調整としてナパポが変更されます。
