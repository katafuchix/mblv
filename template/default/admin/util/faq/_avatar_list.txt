アバター管理
+ アバター管理について
アバターに係る各種パラメータを設定できます。

+ アバターシステムカテゴリとは何ですか？
システム側で予め定義されているアバターの種類・区分のことです。管理者が変更することはできませんので、変更が必要な場合はシステム管理会社にご連絡ください。

+ アバターカテゴリとは何ですか？
管理者が任意に設定できるアバターの種類のことです。アバターを種類別に区分け・分類するために指定します。

+ アバターカテゴリ優先度とは何ですか？
アバターカテゴリのユーザ画面に表示される順番のことです。「0」以上の整数で指定します。数字が小さい順に画面の上から表示されます。「0」の場合は表示されません。

+ 「アバター画像1」と「アバター画像2」の違いは何ですか？
アバターは「アバター画像1」と「アバター画像2」の2つの画像を重ね合わせて指定することができます。「アバター画像1」が「アバター画像2」の上重なり合わされます。
1つの画像のみの場合は、「アバター画像1」のみを指定します。
例えば、「髪の毛」のアバターは「アバター画像1」の前髪と「アバター画像2」の後ろ髪とで指定することができます。

+ 「Z座標」とは何ですか？
アバターの画像を指定する際の配置位置のことです。整数で指定します。アバターは画像を立体的な位置で指定することができ、「0」を起点に数字が大きいほど手前に大きく表示されます。

+ アバター消費ポイントとは何ですか？
ユーザがアバターを購入するときに必要となるポイント数のことです。整数で指定します。ユーザはこのポイント数以上を保持していなければ購入することができません。

+ アバターカテゴリIDとは何ですか？
アバターカテゴリ毎に指定されているIDのことです。アバターカテゴリ登録時にシステム側でユニークな整数が付与されます。管理画面からの変更はできません。

+ アバター性別タイプとは何ですか？
アバターに対する性別タイプのことです。男女（男女共通）、男のみ（男性向け）、女のみ（女性向け）のいずれかを指定します。
例えば、「ドレス」のアバターは「女のみ」のアバター性別タイプを指定することができます。

+ デフォルトアバターとは何ですか？
チェックボックスを有効にして、アバターシステムカテゴリのデフォルトとなるアバターを指定します。
例えば、「街中」のアバターをアバターシステムカテゴリ「背景」のデフォルトとして指定することができます。

+ アバター配信終了数とは何ですか？
アバター配信を終了する最大数のことです。指定された数まで配信を行い、この数を過ぎるとユーザは購入できなくなります。

+ アバター配信終了数設定とは何ですか？
チェックボックスを有効にして、整数で指定することにより、アバター配信終了数を設定します。

+ アバター配信終了日時とは何ですか？
アバター配信を終了する日時のことです。指定された日時まで配信を行い、この日時を過ぎるとユーザは購入できなくなります。

+ アバター配信終了日時設定とは何ですか？
チェックボックスを有効にして、年月日時分秒を「0000-00-00 00:00:00」の形式で指定することにより、アバター配信終了日時を設定します。

+ アバターポータルCMSとは何ですか？
アバターのTOPページ（アバターポータル）全体を編集する機能のことです。アバターポータルの上部と下部それぞれに任意のデザインをHTML形式で指定します。

+ アバターポータルCMSの端末振り分けとは何ですか？
【携帯サイト構築プラットフォーム】では機能しません。指定しても何も作用しません。

+ ファイル管理とは何ですか？
「【携帯サイト構築プラットフォーム】 ファイルアップロード管理画面」が別ウィンドウで起動されます。フリーページに記載したい画像ファイルのアップロードやタグの表示ができます。
