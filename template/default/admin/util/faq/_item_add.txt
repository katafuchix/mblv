商品登録
+ 商品登録について
商品情報の追加を行うことができます。

+ 商品識別IDとは何ですか？
商品を一意に識別できるようにするための、管理者が任意に設定できるIDです。

+ 商品カテゴリとは何ですか？
管理者が任意に設定できる商品の種類のことです。商品を種類別に区分け・分類するために指定します。

+ 仕入先設定とは何ですか？
商品を発注する業者を指定することができます。仕入先設定に表示される情報の管理は「EC管理」→「仕入先管理」から行うことができます。

+ 送料設定とは何ですか？
商品を発送する際の送料を指定することができます。送料設定に表示される情報の管理は「EC管理」→「送料管理」から行うことができます。

+ 決済方法とは何ですか？
商品を購入する際に選択可能な決済方法を指定することができます。

+ 代引き手数料設定とは何ですか？
商品を購入する際に決済方法を「代引き」した場合の代引き手数料を指定することができます。代引き手数料設定に表示される情報の管理は「EC管理」→「代引き手数料設定管理」から行うことができます。

+ 同梱不可フラグとは何ですか？
同時に複数の商品を購入した場合に、同梱発送が可能かどうかを指定することができます。ここにチェックを入れることにより、単一発送の商品として扱われます。

+ 価格とは何ですか？
商品の販売価格を設定することができます。

+ 商品ラベルとは何ですか？
商品情報ページに表示する文言を設定することができます。

+ ポイントとは何ですか？
商品を購入した際に取得できるポイントの設定を行うことができます。

+ キャッチコピーとは何ですか？
商品情報ページに表示するキャッチコピーを設定することができます。

+ 詳細ページに表示する商品情報とは何ですか？
商品詳細ページに表示する商品情報を設定することができます。

+ 商品説明とは何ですか？
商品情報ページに表示する商品説明を設定することができます。

+ 商品スペックとは何ですか？
商品スペックページに表示する商品スペックを設定することができます。

+ 商品画像とは何ですか？
商品ページに表示する商品画像を設定することができます。

+ タイプとは何ですか？
商品タイプ（色・サイズ等）の設定を行うことができます。
タイプは複数設定することができ、タイプを空にすることで指定したタイプを無効にすることができます。

+ 在庫数とは何ですか？
設定したタイプの商品について、在庫数の設定を行うことができます。
在庫を「0」にすることで、指定したタイプの商品を売り切れ状態にすることができます。

+ 優先度とは何ですか？
設定したタイプの商品について、表示順の設定を行うことができます。

+ サンプル画像とは何ですか？
商品画像ページに表示する画像とそのタイトルの設定を行うことができます。
「フォームを追加」ボタンを押下することで、表示する画像を追加することができます。

+ 販売期間設定とは何ですか？
期間限定で販売する商品について、販売期間設定を適応するかどうかの設定を行うことができます。
無効　…　販売期間設定をチェックせずに商品情報を表示
有効　…　販売期間設定をチェックし、startからend内の期間のみ商品情報を表示

+ 販売期間とは何ですか？
販売期間設定を「有効」に設定した場合に、その商品の表示期間を設定することができます。
start　…　販売期間開始日付
end　　…　販売期間終了日付

+ 表示設定フラグとは何ですか？
商品情報を表示するかどうかの設定を行うことができます。
表示　　…　商品一覧ページ・商品ページに商品を表示する
非表示　…　商品一覧ページ・商品ページに商品を非表示しない

+ 商品フリースペースとは何ですか？
商品ページのフリースペース部分を自由に編集することができます。
HTMLによって記述されたソースに規定のタグを挿入することで、画像や商品ページへのリンクを表示することができます。
