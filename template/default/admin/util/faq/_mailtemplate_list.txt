メールテンプレート管理
+ メールテンプレート管理について
システムから送信するメールのひな形をメールテンプレートとして管理を行うことができます。

+ メールテンプレート名とは何ですか？
メールの種類毎に指定するテンプレートのことです。以下の通りです。
会員登録メール　　　　　　　　　…　会員登録時に送信
会員登録時既に会員エラーメール　…　会員登録時にエラーがある（既に会員登録済）場合に送信
メールアドレス変更完了メール　　…　メールアドレス変更の完了時に送信
メールアドレス変更エラーメール　…　メールアドレス変更時にエラーがある場合に送信
会員登録完了メール　　　　　　　…　会員登録の完了時に送信
パスワード請求メール　　　　　　…　パスワード請求時に送信
友達招待メール　　　　　　　　　…　友達招待時に送信
メッセージ受信通知メール　　　　…　メッセージ受信時に送信
友達申請メール　　　　　　　　　…　友達申請時に送信
伝言板書き込み通知メール　　　　…　伝言板書き込み時に送信
画像投稿完了メール　　　　　　　…　画像投稿完了時に送信
画像投稿失敗メール　　　　　　　…　画像投稿時にエラーがある場合に送信

+ メールテンプレートタイトルとは何ですか？
メールの件名のことです。「SNS名の挿入」等の各種独自タグを本文中に設定することができます。

+ メールテンプレート本文とは何ですか？
メールの本文のことです。テキスト形式で指定します。「ログインURLの挿入」等の各種独自タグを本文中に設定することができます。
