+ 通報管理について
ユーザが特定のユーザに対して行う通報の検索や監視を行うことができます。

+ 通報ステータスとは何ですか？
実際に有効となっているか無効となっているかの状態のことです。管理者が有効なものを無効にすることができます。無効なものを有効に復活することはできません。

+ 通報IDとは何ですか？
通報に指定されているIDのことです。通報作成時にシステム側でユニークな整数が付与されます。管理画面からの変更はできません。

+ 通報監視とは何ですか？
管理者が通報の内容に問題がないかの確認を行うことです。確認ができた通報については「通報監視ステータス」を更新することによって、確認した記録を残すことができます。

+ 通報メッセージとは何ですか？
通報登録処理を行ったユーザが通報の際に入力したメッセージのことです。
