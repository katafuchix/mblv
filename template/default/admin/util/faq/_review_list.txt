レビュー管理
+ レビュー管理について
【携帯サイト構築プラットフォーム】で商品を購入したユーザが、その商品に対して投稿したレビュー記事について、編集や削除を行うことができます。

+ レビューとは何ですか？
ユーザが商品を購入するとその商品に対するレビュー記事を投稿することができます。レビューの投稿はレビュー依頼メールに記載されたURLからのみ行うことができます。

+ レビュー依頼メールとは何ですか？
レビュー投稿画面へアクセスするためのURLが記載されたメールのことです。レビュー依頼メールは毎日13:00にチェックが行われ、配信対象となるユーザに自動配信されます。

+ レビュー依頼メールの配信条件は？
商品発送ステータスが「発送済み」に変更された受注情報が対象となります。

+ レビューを検索するには？
ユーザID、ニックネーム、商品名、レビュータイトル、レビュー本文、レビューステータス、レビュー投稿期間を指定して「検索」ボタンを押下することでレビューを検索することができます。

+ レビューステータスの削除待ち、cron待ち、ユーザ投稿待ち、有効とはそれぞれどのような状態ですか？
以下の通りです。
削除　　　　　　…　レビュー投稿が削除された状態
cron待ち　　　　…　レビュー投稿依頼メールの配信待ち状態
ユーザ投稿待ち　…　レビュー投稿依頼メールが配信されユーザからの投稿待ち
有効　　　　　　…　レビュー投稿が完了した状態

+ レビュー情報を編集するには？
対象となるレビュー情報の「編集」リンクを押下します。

+ レビュー情報を削除するには？
対象となるレビュー情報の「削除」リンクを押下します。
