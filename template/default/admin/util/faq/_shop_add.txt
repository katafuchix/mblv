ショップ登録
+ ショップ登録について
ショップ全体の基本的な情報として、共通の名称、色等の各種設定を行うことができます。

+ ショップ名とは何ですか？
各ショップを判別するための店舗名称のことです。

+ ショップニュースとは何ですか？
ショップ一覧に表示するニュース本文のことです。

+ ショップ背景色とは何ですか？
ページの背景色の設定のことです。

+ ショップ文字色とは何ですか？
ページ内に記述された文字の色の設定のことです。

+ ショップリンク色とは何ですか？
ページ内に記述されたリンクの色の設定のことです。

+ ショップアクティブリンク色とは何ですか？
リンクを設定したテキストをクリックした時の色の設定のことです。

+ ショップ訪問済みリンク色とは何ですか？
一度訪問したリンクを表示する時の色の設定のことです。

+ おすすめ商品とは何ですか？
ショップTOPページのおすすめ商品一覧に表示される商品を指定することができます。表示したい順番で商品IDを「,」区切りで入力します。

+ 人気ランキングとは何ですか？
ショップTOPページの人気ランキング一覧に表示される商品を指定することができます。表示したい順番で商品IDを「,」区切りで入力します。

+ ショップ画像1(TOP用)とは何ですか？
マーケットTOPページに表示されるショップ一覧の画像を指定することができます。

+ ショップ画像2(SHOP用)とは何ですか？
ショップTOPページ上部に表示される画像を指定することができます。

+ ショップTOPカテゴリ一覧表示設定とは何ですか？
ショップTOPページにカテゴリ一覧を表示するかどうかの設定を行うことができます。

+ ショップフリースペースとは何ですか？
ショップTOPページのフリースペース部分を自由に編集することができます。
HTMLによって記述されたソースに規定のタグを挿入することで、画像や商品ページへのリンクを表示することができます。
