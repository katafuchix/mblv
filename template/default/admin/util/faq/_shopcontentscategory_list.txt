ショップフリーページカテゴリ管理
+ ショップフリーページカテゴリ管理について
各ショップのフリーページとして独自の画面を管理画面から設定することができます。

+ コンテンツIDとは何ですか？
ショップフリーページ毎に指定されているIDのことです。ショップフリーページ登録時にシステム側でユニークな整数が付与されます。管理画面からの変更はできません。

+ ショップフリーページタイトルとは何ですか？
ショップフリーページのタイトルとなる文言です。

+ ショップフリーページ本文とは何ですか？
ショップフリーページに表示する本文を自由に編集することができます。
HTMLによって記述されたソースに規定のタグを挿入することで、画像や商品ページへのリンクを表示することができます。

+ ショップフリーページを検索するには？
タイトル、本文を指定して「検索」ボタンを押下することでショップフリーページを検索することができます。

+ ショップフリーページカテゴリとは何ですか？
管理者が任意に設定できるショップフリーページの種類のことです。作成されたショップフリーページは必ず1つ以上のカテゴリに分類されます。

+ ショップフリーページフッターリンク設定とは何ですか？
ショップフリーページのフッターに表示するリンクをどのショップページにするか設定することができます。

+ ショップフリーページフッターリンク設定ステータスとは何ですか？
ショップフリーページのフッターに表示するリンクを有効にするかどうかを設定することができます。
有効　…　フッターにリンクを表示する
無効　…　フッターにリンクを表示しない

+ ショップフリーページを追加するには？
ショップフリーページ一覧の「ショップフリーページ新規登録」リンクを押下します。

+ ショップフリーページを編集するには？
ショップフリーページ一覧にて該当するショップフリーページの「編集」リンクを押下します。

+ ショップフリーページを削除するには？
ショップフリーページ一覧にて該当するショップフリーページの「削除」リンクを押下します。
