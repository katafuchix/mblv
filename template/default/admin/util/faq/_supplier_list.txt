仕入先管理
+ 仕入先管理について
商品情報の設定に使用する仕入先情報の設定を行うことができます。

+ 仕入先を追加するには？
仕入先一覧の「仕入先設定新規登録」を押下します。

+ 仕入先を編集するには？
仕入先一覧にて該当する仕入先の「仕入先編集」リンクを押下します。

+ 仕入先を削除するには？
仕入先一覧にて該当する仕入先の「仕入先削除」リンクを押下します。

+ 仕入先名とは何ですか？
仕入先メーカーの名称を設定することができます。

+ 出荷タイプとは何ですか？
商品の出荷元を設定することができます。
直接ユーザへ出荷　…　メーカーからユーザへ直送
納品後に出荷　　　…　倉庫に納品された後ユーザへ出荷

+ 送料設定とは何ですか？
商品を発送する際の送料を指定することができます。
送料設定に表示される情報の管理は「EC管理」→「送料管理」から行うことができます。

+ 代引き手数料設定とは何ですか？
商品を購入する際に決済方法を「代引き」した場合の代引き手数料を指定することができます。代引き手数料設定に表示される情報の管理は「EC管理」→「代引き手数料設定管理」から行うことができます。

+ 同梱可能設定とは何ですか？
同梱可能な仕入先を指定することができます。
