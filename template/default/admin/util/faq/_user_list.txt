+ ユーザ検索について
【携帯サイト構築プラットフォーム】に登録されている会員に係る設定等の操作を行うことができます。

+ ステータスの仮登録、本登録、退会、強制退会とはそれぞれどのような状態ですか？
以下の通りです。
仮登録　　…　会員登録メールが送信されている（会員登録が完了していない）状態
本登録　　…　会員として登録されている状態
退会　　　…　ユーザから退会されている状態
強制退会　…　運用者から強制的に退会されている状態

+ ユーザの強制退会はどのように行いますか？
ユーザ情報編集画面の一番下の「強制退会する」を押下します。

+ 強制退会処理をすると、ユーザには通知されますか？
通知されません。

+ 強制退会処理したユーザを本登録に復活することはできますか？
できません。新たに新規の会員として登録する必要があります。

+ エラーメール数とは何ですか？
システムから送信されたメールがエラーで返却された回数のことです。

+ ユーザの属性項目（趣味、等）を増やすことができますか？
基本設定のプロフィール管理から行うことができます。ただし、ユーザ検索の検索項目とすることはできません。

+ ゲストユーザとは何ですか？
著名人等向けのユーザのことで、携帯電話からだけでなくPCからユーザ画面に入力可能とします。初回のユーザ登録は通常通り携帯電話から行います。

+ ユーザのログイン履歴を見ることはできますか？
できません。

+ ユーザを管理画面から新規作成することはできますか？
できません。ユーザが携帯電話から会員登録を行うことでユーザを新規作成します。

+ ユーザの登録情報を修正したい。
ユーザ検索画面で該当ユーザのニックネームを押下します。
